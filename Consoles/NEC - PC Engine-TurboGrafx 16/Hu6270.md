## HU 6270 Netlist

### Conventions used



| Short | Long          | Short | Long             |
| ----- | ------------- | ----- | ---------------- |
| Dxx   | Data (number) | Axx   | Address (number) |
| /WR   | Write Enable  | /OE   | Output Enable    |
| /CE   | Chip Enable   | NC    | Not Connected    |
| I/O   | Input/Output  | RAxx  | RAM pin#         |
